运行时常用方法:

  1. objc_allocateClassPair

    Creates a new class and metaclass:

    Class objc_allocateClassPair(Class superclass, const char *name, size_t extraBytes);
/**
1. superclass
The class to use as the new class's superclass, or Nil to create a new root 
class.
2. name
The string to use as the new class's name. The string will be copied.
3. extraBytes
    The number of bytes to allocate for indexed ivars at the end of the 
    class and metaclass objects. This should usually be 0.
*/

    例如:
    Class newClass = objc_allocateClassPair([NSError class], "TestClass", 0);
    运行时创建了一个NSError的子类TestClass

  2. class_addMethod

    Adds a new method to a class with a given name and implementation.
    type:几种取值查看

    BOOL class_addMethod(Class cls, SEL name, IMP imp, const char *types);
/**
cls: The class to which to add a method.
name: A selector that specifies the name of the method being added
imp: A function which is the implementation of the new method. The function 
must take at least two arguments—self and _cmd.
types: An array of characters that describe the types of the arguments to the 
method. the second and third characters must be “@:” (the first character is 
the return type).

return:成功YES 失败NO
*/

    栗子:
    class_addMethod(newClass, @selector(testMetaClass), (IMP)TestMetaClass, "v@:");
    为这个新类添加一个方法testMetaClass，这个方法的实现是TestMetaClass函数
